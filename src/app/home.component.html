A platform for remote data collection, data storage, and a data analysis framework that uses web, mobile, and cloud technologies for the detection and forecasting of plant diseases.

Features


Cloud Platform

The cloud platform stores and processes all the data that is used in the mobile and web platforms.

CITAS Database

The database contains all data pertinent for the system such as plant information, soil conditions, and geospatial data.





Mobile Platform

The mobile platform is used by the farmers on the field to collect data from their respective farms. Components:

  • Data Capture (TANIM)
  • Image Capture
  • Image Analysis
  • Data Analysis
  • Visualization

Plantscope

The PlantScope is composed of a mobile microscope paired with software for image processing, geospatial tagging, and QR coding for automated and easy labeling of plants.

Soil Data Collection

Wireless Sensors deployed around the farm sites collect environmental data parameters and are retrieved by the farmers via the mobile app.

LAMP Assay Kit

The LAMP Assay Kit is composed of a thermocycler and reagents that is used for the molecular detection of Foc TR4.

Web Platform

The web platform is accessible to researchers who wish to acquire data gathered by the farmers and analyzed by the system.

Soil Analysis

Soil parameters gathered from the field are plotted on a graph to show the trends over a given time period.

Geospatial Analysis & Disease Mapping

The web platform visualizes the spread of Foc TR4 in the respective farm sites and analyze where the disease might spread to next.

Plant Image Analysis

Images loaded into the system are processed and analyzed to see if the plants are infected with Foc TR4 or not.
